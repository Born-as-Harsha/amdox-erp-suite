# Stakeholder Requirements

## Introduction

Every software project is developed for specific users called stakeholders. A stakeholder is any person or group who will use the software or benefit from it.

For this ERP project, the following stakeholders have been identified based on the project requirements.

---

# Stakeholder 1 – Company Executives

### Requirements

* View company performance dashboard.
* Monitor business KPIs.
* Access reports.
* View overall business growth.

### Priority

High

---

# Stakeholder 2 – Finance Department

### Requirements

* Manage financial records.
* Track income and expenses.
* Generate invoices.
* Record transactions.
* Generate financial reports.

### Priority

High

---

# Stakeholder 3 – Human Resource Department

### Requirements

* Employee registration.
* Attendance management.
* Leave management.
* Payroll information.
* Employee records.

### Priority

High

---

# Stakeholder 4 – Inventory Manager

### Requirements

* Product management.
* Inventory tracking.
* Low stock alerts.
* Purchase records.
* Supplier information.

### Priority

Medium

---

# Stakeholder 5 – Project Manager

### Requirements

* Create projects.
* Assign tasks.
* Track project progress.
* View deadlines.
* Generate project reports.

### Priority

Medium

---

# Stakeholder 6 – System Administrator

### Requirements

* Manage users.
* Assign roles.
* Configure system settings.
* Manage permissions.
* Monitor system security.

### Priority

High

---

# Functional Requirements

The system should provide:

* Secure Login
* User Registration
* Dashboard
* Finance Module
* HR Module
* Inventory Module
* Reports
* Notifications

---

# Non-Functional Requirements

The system should also be:

* Secure
* Easy to use
* Fast
* Reliable
* Scalable
* Responsive on different screen sizes

---

# Summary

Understanding stakeholder requirements helps ensure that every module developed during the internship solves a real business need. These requirements will guide the design and implementation of the ERP system.
